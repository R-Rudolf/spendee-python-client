Create a budget

Parameters

  • name (str) – name of the budget

  • limit (str) – the budget amount

  • currency (str) – amount’s currency

  • wallets (list) – list of wallets this budget is relevant to

  • categories (list) – list of categories this budget is relevant to

  • users (list) – list of users this budget is shared with

  • position (int) – order position of the budget among all the budgets

  • start_date (Optional[date]) – when

  • end_date (Optional[date]) – when is the end of the budget. Only relevant when period=’once’

  • status (str) – active

  • period (str) – once/daily/weekly/biweekly/monthly/yearly

  • offline (bool) – whether it was created offline

  • notification (bool) – whether to enable notifications for this budget

  • all_categories_selected (bool) – whether all categories were selected or only a subset of them

  • all_wallets_selected (bool) – whether all wallets were selected or only a subset of them

  • all_users_selected (bool) – whether all users were selected or only a subset of them
